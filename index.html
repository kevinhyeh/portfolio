The Graphic Jar is a graphic design freelance that a college friend, Christina Sayegh, and I started in January 2017. We specialize in designs that involve branding a business, such as logos, business cards, and websites. Christina would handle all of the illustration aspects of the freelance where I would take care of any development needed.

Technologies used for the website includes HTML, CSS / SASS, Javascript and a bit of Bootstrap. The website is mobile friendly where the use of media queries and javascript were needed to change the layout of the website and navigation bar.

Another aspect of the website includes a project builder application that I have developed using ReactJS, NodeJs, and Express; also Now for deployment (link above). This application allows users to create a simple starting point for their desired project with a bit of customization. Their project is then sent to us via email where we can start the project from there.

Deck Challenge

Github: https://github.com/kevinhyeh/tgj_website

The Deck Challenge is a mobile application that was developed for the final project during my bootcamp. It is a workout base application using a deck of playing cards as your instructor. Shown in the demo video, the workouts that you select to incorporate in your workout is randomly assigned to either red or black cards (if 2 workouts are chosen) or each of the different suits (if 4 workouts are chosen).

Technologies used are React Native, NodeJS, and MYSQL.

This was my first time using React Native and I absolutely enjoyed it. There are parts where I can improve, which I plan on doing as time goes by. Also plan on deploying the app onto Apple's App Store in the near future. It is quite a simple app but there are features that can be incorporated to make it excel.

UCBE Forum

Github: https://github.com/BryantFukushima/UCBE-coding-bootcamp-forum

The UCBE Forum is a collaborative application that was developed between a friend, Bryant Fukushima, and I for the second project during my bootcamp. The app is a basic forum where past, present, and potential future students of UC Berkeley's Coding Bootcamp can use to get inquiries about the class. With a login system, people can create posts, leave comments, and like or dislike posts / comments.

Technologies used are EJS, NodeJS, and MYSQL.

Even though this was a collaborative project, Bryant and I had identical knowledge of every aspect of the code, including the SQL queries.
