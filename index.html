The Graphic Jar

Website: https://kevinhyeh.github.io/tgj_website/

Project Builder App: https://tgj-project-builder-q98mnwxq6.now.sh/

Github: https://github.com/kevinhyeh/tgj_website

The Graphic Jar is a graphic design freelance that a college friend, Christina Sayegh, and I started in January 2017. We specialize in designs that involve branding a business, such as logos, business cards, and websites. Christina would handle all of the illustration aspects of the freelance where I would take care of any development needed.

Technologies used for the website includes HTML, CSS / SASS, Javascript and a bit of Bootstrap. The website is mobile friendly where the use of media queries and javascript were needed to change the layout of the website and navigation bar.

Another aspect of the website includes a project builder application that I have developed using ReactJS, NodeJs, and Express; also Now for deployment (link above). This application allows users to create a simple starting point for their desired project with a bit of customization. Their project is then sent to us via email where we can start the project from there.
